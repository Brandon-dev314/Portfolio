# Brandon Eroza — Portfolio

A personal portfolio website with an integrated AI chatbot assistant (DevAgent), built with React and powered by OpenAI.

**Live:** [https://portfolio-beige-six-57.vercel.app](#) · **Backend:** Hosted on Railway

---

## Overview

A dark-themed, minimalist portfolio featuring smooth scroll-reveal animations, a light/dark mode toggle, and a floating AI chatbot that answers questions about my background, projects, and tech stack in real time.

## Tech Stack

### Frontend
- React 18 + Vite
- CSS Variables for theming (dark/light mode)
- Sora + DM Mono typography
- IntersectionObserver-based scroll animations
- Deployed on **Vercel**

### Backend
- Node.js + Express
- OpenAI API (GPT-4o-mini)
- Single endpoint proxy to protect API keys
- Deployed on **Railway**

## Features

- **Responsive design** — works on desktop, tablet, and mobile
- **Dark / Light mode** — full theme toggle with smooth transitions
- **Scroll animations** — staggered reveal on each section via IntersectionObserver
- **AI Chatbot (DevAgent)** — floating assistant powered by OpenAI that knows about my projects, skills, and experience. Supports English and Spanish
- **Project showcase** — cards with tech tags, GitHub links, and live demo links
